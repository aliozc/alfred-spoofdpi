# alfred-spoofdpi
Simple Alfred workflow to toggle SpoofDPI

![](https://github.com/aliozc/alfred-spoofdpi/blob/main/screenshot.png?raw=true)

# Installing
1. [Download the latest release](https://github.com/aliozc/alfred-spoofdpi/releases/download/1.0.1/SpoofDPI.alfredworkflow)
2. Double-click or drag the file to Workflows page in Alfred Preferences.
3. Set the `spoofdpiPath` variable in the import screen.

# Usage
1. Trigger the keyword `spoof` in Alfred.
2. Start or stop the service by clicking `Enter`
